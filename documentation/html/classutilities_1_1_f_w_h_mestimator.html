int utilities.FWHMestimator.estimate ( self,
int nstars,
float fwhm_max = 25,
int skip_max = 10,
bool full_output = False )

Estimates the FWHM for a given number of stars.

Minimum FWHM is 2. Otherwise it is considered a cosmic and is skipped. This method can only estimate in even steps.

Parameters
nstars(int): number of stars that should be used for the estimation
fwhm_max(float): highest possible value of the fwhm
skip_max(int): maximum number of stars that should be skipped.
full_output(bool):
Returns
(int|list[int], int): estimated FWHM value, or if full_output is set to True a list of estimated FWHM values and the number of skipped stars.
