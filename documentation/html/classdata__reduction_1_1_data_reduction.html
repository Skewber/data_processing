Detailed Description

Basic class for Data Reduction.

For this class to work properly the used .fits files need a the following keywords in the header: 'EXPTIME', 'FILTER', 'OBJECT'. Optionally it can also contain wcs informations if an alignment of those is desired. If not provided the alignemt can also be done by matching stars in the image.

None data_reduction.DataReduction.stack_light ( self,
str alignment = 'none' )

Stacks the light frames and alignes them, if desired.

Parameters
alignment(string) : optional, string that sates the alignment method. Possibilities are 'none' (default), 'star1', 'star2' and 'wcs'. Otherwise a value Error will be raised. 'none' stacks the iamges without aligning them, 'star1' uses the astroalign library, 'star2' requires a usir input to detect a star, 'wcs' uses the wcs information from the header. Details to 'star2': an image will be shown using matplotlib.pyplot.imshow(). Click on the star, that should be aligned.
